# utils
Папка содержит вспомогательные утилиты на Питоне. 

В настоящий момент в ней содержится две программы, которые помогают работать с .ipynb - файлами

Если вы хотите запустить отладчик, вам может быть удобно сконвертировать ipynb файл в обычный питоновский.
А если нужно включить ту или иную информацию в свою базу знаний, используйте конвертер в md

Обе программы устроены одинокого и максимально просто. Они используют стандартный json - парсер для парсинга ipynb

ipynb2py.py - конвертирует .ipynb файл в .py
ipynb2md.py - конвертирует .ipynb файл в .md

## Использование

Путь к файлу или папке задаётся вручную в тексте программы. Если задан путь к папке, программа конвертирует все файлы .ipynb, которые найдёт в папке, создавая файлы с таким же именем, но расширением .py и .md соответственно.

Будьте аккуратны, файлы .py и .md будут перезаписаны!

Вам возможно понадобится минимальная доработка результата, но вы будете избавлены от необходимости всё копипастить руками. Кроме того, вам может быть интересно посмотреть код этих программ (который максимально прост)
